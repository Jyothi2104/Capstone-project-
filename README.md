# Bluestock Mutual Fund Analytics Platform

## Overview

The Bluestock Mutual Fund Analytics Platform is an end-to-end financial analytics project developed as part of the Bluestock Internship Capstone Program.

The project combines data engineering, financial analytics, risk modeling, and business intelligence techniques to analyze mutual fund performance, investor behavior, portfolio concentration, and market trends.

The platform processes multiple mutual fund datasets, computes advanced performance and risk metrics, and provides interactive visualizations through Power BI dashboards.

---

## Project Objectives

* Build a structured mutual fund analytics database
* Perform exploratory data analysis on mutual fund datasets
* Evaluate fund performance using financial metrics
* Analyze investor transaction patterns
* Compute advanced risk metrics
* Develop a simple fund recommendation system
* Create interactive Power BI dashboards
* Generate actionable business insights

## Key Findings

* Large-cap funds demonstrated stable risk-adjusted performance.
* Small-cap funds delivered higher returns with higher risk.
* SIP participation remained strong across investor segments.
* Investor activity was concentrated in a limited number of regions.
* Portfolio concentration varied significantly across funds.
